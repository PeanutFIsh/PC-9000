PC-9000 Project Page

 

Welcome to the project page of the PC-9000 Project! This includes :

- Diskette Simulation

- BIOS Simulation

- HDD Simulation

- OS Simulation

 

It works like a typical computer, you insert the diskette, install the OS and then you can use it.

In this simulator, you download the OS on my GitHub then you have to install it!
